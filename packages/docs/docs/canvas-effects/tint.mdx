# tint()<AvailableFrom v="4.0.452" />

A 2D [canvas effect](/docs/canvas-effects) that overlays a flat color onto the source. The tint respects the source's alpha mask via `globalCompositeOperation: 'source-atop'`.

```tsx twoslash title="src/TintDemo.tsx"
import {tint} from '@remotion/canvas-effects';
import {Solid} from 'remotion';

export const TintDemo = () => (
	<Solid
		color="hotpink"
		width={1920}
		height={1080}
		effects={[tint({color: 'cyan', amount: 0.5})]}
	/>
);
```

## Parameters

### color

The CSS color to overlay onto the source.

### amount?

Blend strength between `0` (no tint) and `1` (full color over opaque pixels). Values are clamped. Default `0.5`.
