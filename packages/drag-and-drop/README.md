# @remotion/drag-and-drop

Construct and parse drag-and-drop payloads for Remotion

[![NPM Downloads](https://img.shields.io/npm/dm/@remotion/drag-and-drop.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@remotion/drag-and-drop?minimal=true)

## Installation

```bash
npm install @remotion/drag-and-drop --save-exact
```

When installing a Remotion package, make sure to align the version of all `remotion` and `@remotion/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.remotion.dev/docs/drag-and-drop) for more information.
