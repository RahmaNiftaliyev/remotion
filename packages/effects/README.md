# @remotion/effects

**Experimental and undocumented.** Presets for Remotion's canvas effect hooks; behavior and exports may change in any release.

The package is published on npm (`publishConfig.access: public`) for early adopters. There is no stable documentation page; refer to the source under [`packages/effects`](https://github.com/remotion-dev/remotion/tree/main/packages/effects) in the Remotion repo.

## Installation

```bash
npm install @remotion/effects --save-exact
```

Align the version of `remotion` and all `@remotion/*` packages to the same release (pin exact versions in `package.json`).
