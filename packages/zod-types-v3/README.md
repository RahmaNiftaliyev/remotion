# @remotion/zod-types-v3

Zod v3 types for Remotion. Use this package if you are still using Zod v3 and need to compose Remotion schemas into a `z.object()`.

If you are using Zod v4, use `@remotion/zod-types` instead.

## Installation

```bash
npm install @remotion/zod-types-v3
```
